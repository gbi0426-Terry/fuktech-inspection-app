# 富泰空調 巡檢填報系統 — 行動填報原型

依富泰空調科技提供的「FW_帆軟-填報功能-範例」四份單位點檢表製作的手機填報視覺原型：

1. P-T02-05 機具設備每日點檢表（電動堆高機）
2. I-Z01-03 有機溶劑每日點檢表（磁石組立）
3. I-Z01-02A 化學品區域查檢表（每月）
4. AED 日常檢查紀錄表（每月）

本階段僅針對「填報畫面」製作，不含到期提醒/告警功能。純前端 HTML/CSS/JS，無需建置工具，直接開啟 `index.html` 或透過任意靜態伺服器即可預覽。

## 預覽
```
python3 -m http.server 5044
```
開啟 http://localhost:5044
